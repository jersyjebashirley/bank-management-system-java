# Bank Management System (Java)

A simple Bank Management System built using Java with OOP concepts.

## Features
- Deposit money
- Withdraw money
- Transfer money
- Show account details
- Menu-driven program

## Concepts Used
- Abstraction
- Inheritance
- Method Overriding
- Classes and Objects
- Scanner for user input

## How to Run
## How to Run

1. Compile the program:

```bash
javac Main.java
```

2. Run the program:

```bash
java Main
```
